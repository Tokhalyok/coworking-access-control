# 🏗️ Архитектура проекта IoT

## 📊 Общая структура

```
IoT/
├── abi.json                    # ABI смарт-контракта
├── API.py                      # Python API для взаимодействия с блокчейном
├── build.bat                   # Скрипт сборки для Windows
├── CoworkingAccess.sol         # Smart контракт на Solidity
├── README.md                   # Документация проекта
├── Run.py                      # Точка входа (запуск приложения)
├── blockchain/                 # Компонент блокчейна
│   ├── abi.json               # ABI контракта
│   └── CoworkingAccess.sol    # Smart контракт
├── desktop/                    # Десктоп приложение (Python)
│   ├── API.py
│   ├── build.bat
│   └── Run.py
└── web/                        # Веб приложение (React)
    └── coworking-react/        # React приложение
        ├── package.json
        ├── vite.config.js
        ├── tailwind.config.js
        ├── postcss.config.js
        ├── public/
```

## 🔵 Описание компонентов

### 📁 Корневая папка IoT

**Назначение:** Главная папка проекта, содержит основные конфигурационные файлы и скрипты запуска.

| Файл | Описание |
|------|---------|
| `abi.json` | Application Binary Interface смарт-контракта для взаимодействия с блокчейном |
| `CoworkingAccess.sol` | Smart контракт на Solidity для управления доступом в коворкинг |
| `API.py` | Python API для работы со смарт-контрактом |
| `Run.py` | Главный скрипт для запуска приложения |
| `build.bat` | Скрипт сборки для Windows систем |
| `README.md` | Документация и инструкции по использованию |

---

### 🔴 blockchain/ — Блокчейн слой

**Назначение:** Содержит смарт-контракты и их интерфейсы для взаимодействия с блокчейном.

| Файл | Описание |
|------|---------|
| `CoworkingAccess.sol` | Основной контракт для управления доступом в коворкинг |
| `abi.json` | Интерфейс контракта для взаимодействия из приложений |

**Функциональность:**
- Управление правами доступа пользователей
- Регистрация пользователей
- Логирование входов и выходов
- Хранение информации о доступе

---

### 🟠 desktop/ — Десктоп приложение

**Назначение:** Python приложение для локального запуска на Windows с интеграцией смарт-контракта.

| Файл | Описание |
|------|---------|
| `API.py` | API для обработки запросов и работы с контрактом |
| `Run.py` | Запуск десктоп приложения |
| `build.bat` | Скрипт сборки приложения |

**Функциональность:**
- Локальное управление доступом
- Прямое взаимодействие со смарт-контрактом
- Управление пользователями и логами

---

### 🟢 web/coworking-react/ — Веб приложение

**Назначение:** React приложение с использованием современного стека (Vite, Tailwind CSS).

| Файл/Папка | Описание |
|------|---------|
| `package.json` | Зависимости проекта |
| `vite.config.js` | Конфигурация Vite (сборщик) |
| `tailwind.config.js` | Конфигурация Tailwind CSS |
| `postcss.config.js` | Конфигурация PostCSS |
| `public/` | Статические файлы |
| `src/` | Исходный код React компонентов |

**Стек технологий:**
- ⚛️ React
- ⚡ Vite (быстрая сборка)
- 🎨 Tailwind CSS (утилит-фёрст стилизация)
- 📱 Responsive design

## 🎯 Архитектурная концепция

Проект использует **многоуровневую архитектуру** с разделением на три основных слоя:

### 1️⃣ Smart Contract Layer (Блокчейн)
- Смарт-контракт `CoworkingAccess.sol` на Solidity
- Хранение и управление данными доступа
- Гарантия неизменяемости и прозрачности

### 2️⃣ Backend Layer (API)
- Python API для взаимодействия со смарт-контрактом
- Доступен как для десктоп, так и для веб приложений
- Обработка бизнес-логики

### 3️⃣ Frontend Layer (UI)
- **Web UI:** React приложение с современным интерфейсом
- **Desktop UI:** Python приложение для локального управления
